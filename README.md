# Performance-Commerciale-Analyse-du-Portefeuille-Clients
Je viens de finaliser un dashboard complet permettant d’analyser la performance commerciale d’un portefeuille bancaire : PNB, équipement produit, structure client et contribution des agences.
Objectifs du dashboard
Donner une vision claire du PNB global et par produit (Crédit, MRH, Épargne)
Identifier les opportunités commerciales via les taux d’équipement
Comprendre la structure du portefeuille (âge, type de crédit, segmentation)
Comparer la performance des agences pour orienter les actions managériales
Indicateurs clés intégrés
- Nombre de clients
- PNB total et PNB moyen par client
- PNB par produit
- Taux d’équipement crédit & produit
- Répartition des clients par tranche d’âge et type de crédit
- PNB par agence
🛠️ Technos utilisées
Power BI (DAX, Power Query, modélisation)
Nettoyage & transformation des données
Construction d’un thème visuel harmonisé.
